修改VS code HTML的lang

本文最后更新于：2022-7-23

前言

在使用VS code开发前端的时候输入英文"!"快捷生成HTML骨架

但是骨架上带的 lang 是 en

（lang="en" 这个意思是告诉游览器这个页面用的是英语,游览器上的自动翻译就是靠这个）

本文就是修改 en 成为 zh-CN

修改前

<html lang="en">

修改后

<html lang="zh-CN">

修改步骤

一. 打开VS code 并打开设置(位于屏幕左下角)

二. 搜索emmet.variables

三. 点击下面的添加项，把默认的en修改成zh-CN

结尾

做到第三步就OK了，可以去试试看
